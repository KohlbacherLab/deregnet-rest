getGraphs

Returns a list of all available graphs

URL

https://virtserver.swaggerhub.com/sebwink/deregnet/1.0.0/graphs

HTTP Method

GET

Response Type

List

Parameters

  • Query: searchString String

    pass an optional search string for narrowing the list

  • Query: skip Integer

    number of records to skip for pagination

  • Query: limit Integer

    maximum number of records to return

HTTP Method

POST

Response Type

Parameters

  • Body: initalGraphInfo InitalGraphInfo(InitalGraphInfo)
